# Code Php Dasar 2
```
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>PHP Dasar</title>
</head>

<body>
  <!-- Variabel $_GET -->
  <h2><b>Predefine Variable</b></h2>
  <?php
  $nama = 'Kevin';
  echo 'Selamat Datang ' . $_GET['nama']
  ?>
</body>
</html>
```
